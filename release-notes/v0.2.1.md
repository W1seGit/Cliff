# Cliff v0.2.1

This patch improves installed dashboard behavior and makes server usage charts more truthful.

## Fixes

- Fixed copy buttons failing when Cliff is opened from a LAN IP or hostname over HTTP by adding a browser clipboard fallback.
- Removed smoothing from server usage charts so CPU, memory, and player history render as straight measured segments.
- Kept usage chart gaps separated so stopped periods do not look like continuous server activity.
